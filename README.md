# Real_Time_Attendance_Tracker
# Smart Attendance System

A **real-time face recognition system** for automating student attendance tracking using **DeepFace**.  
This project uses **computer vision** to detect and recognize student faces via webcam and automatically mark their attendance in a database.

---

## Tech Stack

**Frontend:** React  
**Backend:** Flask (Python)  
**Libraries:** DeepFace, OpenCV  
**Database:** MySQL  
**Language:** Python, JavaScript
